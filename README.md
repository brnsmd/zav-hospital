# Zav (Завідувач)

Система управління відділенням кістково-гнійної травматології військового госпіталю.

## Статус: Заморожено (планування завершено)

> Проект очікує призначення завідувачем відділення.
> Планування завершено 2024-12-02.

## Ключові рішення

### Інфраструктура
- В ClickUp тільки № історії — ПІБ підтягується з CyberIntern
- **Telegram** — основний інтерфейс (лікарі, пацієнти, завідувач)
- CyberIntern multi-user + Claude API
- Витрати Claude API: ~$18-25/місяць

### Модуль завідувача (locked)
- **CLI** — для складних запитів
- **Тижневий оперплан** (ПН-ПТ): подання до п'ятниці → план на тиждень
- **Екстрені операції** — на наступний день (не в той же день)
- **План госпіталізацій** — AI перевіряє overflow → завідувач затверджує
- **Евакуаційний потік** — сортування, терміново/планово
- **Контроль виписки** — інше відділення / інша лікарня / частина / реабцентр
- **ВЛК workflow** — моніторинг черги, категорії придатності
- **Етапне лікування** — моніторинг повернень на наступні етапи
- **"Залежані" пацієнти** — AI визначає, рекомендує дії
- **Навантаження на лікарів** — розподіл пацієнтів
- **Статистика для командування** — щомісячні звіти

### Медичний контроль
- **Антибіотикотерапія** — посів, антибіотикограма, тривалість, MRSA/ESBL
- **Облік апаратів** — фіксатори (Ілізарова, стрижневі), VAC, дренажі
- **Система алертів** — критичні (t°, СРБ), попередження, інформаційні

### CyberIntern інтеграція
- **Щоденники** — хто написав / не написав (по лікарях)
- **Прострочені аналізи** — не взято / не переглянуто
- **Консультації** — запит / виконано / рекомендації
- **МВТН** — новий / закінчується / продовження
- **Призначення** — нове / змінено / не виконано

### Інші модулі
- **"The Doctor Is In"** — автоматичне відстеження доступності лікарів

## Етапи (після призначення)

1. **Тиждень 1:** Налаштувати ClickUp вручну
2. **Місяць 1:** Працювати вручну, збирати feedback
3. **Місяць 2:** MVP Telegram бота (секретар + тижневий оперплан)
4. **Місяць 3:** Режим лікарів + пацієнтів
5. **Місяць 4:** CyberIntern → Claude API + multi-user
6. **Місяць 5:** "The Doctor Is In" + публічна сторінка
